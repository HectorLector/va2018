Group 3 - "Visualization Wizards"
Topic: Document Cards
-----------------------------------------------

How to run the preprocessing code:

1.) Go in folder "preprocessing"
2.) Add your pdf file(s) in folder: \preprocessing\in
2.) Run command:  .\clean.sh && .\extract.sh && .\transform.sh
3.) You should get a file "visualizer.csv", this can be loaded in Visualizer Tool.
4.) Do not change the data and select all columns/data for visualization.

------------------------------------------------

Tools/Requirements:

 - Linux shell
 - pdfimages, pdftotext, pdfinfo
 - node js (Version 8)
 - npm
 - Natural JavaScript library: "npm install natural"
